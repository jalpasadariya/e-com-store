🛒 E-Com Store - React & Redux Toolkit
A dynamic and responsive E-commerce product gallery built with React.js. This project features real-time product filtering, price sorting, and a fully functional shopping cart managed by Redux Toolkit.

🚀 Live Demo


✨ Features
State Management: Used Redux Toolkit for efficient cart management.

Category Filtering: Filter products by categories like Laptops, Mobiles, Audio, etc.

Price Sorting: Sort products by Price (Low to High & High to Low).

Responsive Design: Optimized for all screen sizes using Tailwind CSS.

Modern UI: Clean and professional interface with React Icons.

🛠️ Tech Stack
Frontend: React.js (Hooks, Functional Components)

State Management: Redux Toolkit (@reduxjs/toolkit)

Styling: Tailwind CSS

Icons: React Icons
